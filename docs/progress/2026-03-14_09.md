## What was done

2. **Magnetic moment predictions**: Modified predict cell to capture CHGNet magnetic moments. Added box plot of |magnetic moment| per atom by constraint type.

3. **SCIGEN vs training comparison**: Predict CHGNet energies for 50 MP-20 training structures. Side-by-side histograms comparing energy and volume distributions (generated vs training).

4. **Before/after relaxation visualization**: 3D matplotlib plot showing structure before and after CHGNet relaxation. 2D displacement arrow plot showing how atoms moved.

5. **E_hull for generated materials**: Compute energy above hull for SCIGEN structures using the test set phase diagram. Histogram of E_hull distribution + box plot by constraint type.

6. **Best candidates table**: Top 15 structures ranked by E_hull. Scatter plot of E_hull vs formation energy colored by magnetic moment (bottom-left = most promising).
